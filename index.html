Start Farming in Minutes

1

Generate Your Wallet

Generate an Ed25519 wallet secured by a 24-word mnemonic. Your wallet address receives block rewards when you win. No KYC, no registration, just pure cryptography.

2

Create Your Plots

Pre-compute cryptographic hashes and store them as plots. Larger plots = more winning chances. Start with k=28 (8.6GB) or k=30 (34GB) for competitive farming.

3

Start Farming

Connect to the network and start checking challenges. Your plots are scanned every 2 seconds. Win blocks, earn 20 RCHV rewards. No continuous energy consumption or waste.

Features

Why Choose Archivas

Sustainable consensus meets enterprise-grade performance. Built for the next generation of blockchain infrastructure.

Energy-Efficient Mining

Mine blocks using disk space instead of burning electricity. No GPUs, no ASICs. Just allocate storage and start earning. Perfect for eco-conscious blockchain participants.

VDF-Powered Security

Verifiable Delay Functions ensure fair block production. Time-based proofs prevent wealthy actors from dominating. Temporal consensus that cannot be parallelized or rushed.

Automatic Sync & Recovery

New nodes sync 100k+ blocks in seconds via IBD. Resume from any height if interrupted. No manual setup. Join the network anytime, anywhere.

Real-Time Observability

Prometheus metrics, Grafana dashboards, and health endpoints. Monitor your node's performance with production-grade tools. 99.9% network uptime tracking.
